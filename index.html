Congressional Tour Tracker

Installable mobile-first tracker for constituent tour requests. Groups are the primary record, with shared action status and shared email generation across all tours in that group.

Install on iPhone: Tap Share, then Add to Home Screen.
This PWA works offline after first load. GitHub Pages alone still cannot send true background iPhone push notifications, so reminders are handled through browser alerts while open and downloadable calendar reminder files.

Quick add group request

One saved group can contain up to 5 tours. Action status, action date, reminders, and email generation all apply to the full group.
